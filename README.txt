run the cshttp program with a port number from the command line like so:
./cshttp 80

then connect to the server from any webbrowser. For example:
http://localhost/servertime or http://remotehostname/servertime

The following commands are supported

/servertime
/login?username=XXX
/logout
/browser
/redirect?url=http://www.websitename.com
/addcart?item=XXX
/delcart?itemnr=XXX
/checkout
